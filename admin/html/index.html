О Системе

Назначение

ORCHID - многоуровневая система защиты на базе ИИ для обнаружения и предотвращения кибератак в реальном времени. Combines ML модели с rule-based детектированием для комплексной защиты.

Как Работает

Захват Трафика

Traffic Sniffer перехватывает HTTP/HTTPS трафик через pyshark для глубокого анализа пакетов

ML Анализ

Isolation Forest обнаруживает аномалии, Random Forest классифицирует тип атаки

DDoS Детектор

Мониторит RPS и паттерны endpoint'ов для выявления DDoS атак

Анализ Эксплойтов

Exploit Analyzer выявляет многоэтапные цепочки атак и паттерны

Автоблокировка

Автоматически блокирует IP после 5 атак в течение 60 секунд

Логирование

Все атаки записываются в зашифрованную SQLite БД через SQLCipher
